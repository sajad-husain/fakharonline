# Fakhar Online - Project Outline

## File Structure

### Core HTML Pages
- **index.html** - Homepage with hero section, product showcase, and seasonal highlights
- **products.html** - Complete fruit catalog with filtering and detailed product information
- **about.html** - Company story, sustainability practices, and farm-to-table journey
- **blog.html** - Educational content, recipes, and farming stories with admin capabilities

### Supporting Files
- **main.js** - Core JavaScript functionality, cart management, and interactive features
- **resources/** - Directory for all images, icons, and media assets

## Interactive Components Implementation

### 1. Dynamic Product Showcase (index.html)
- **Technology:** Splide.js carousel with custom styling
- **Features:** Auto-play, infinite loop, touch/swipe support
- **Data:** JSON-based product information with seasonal availability
- **Interactions:** Hover effects, quick-add to cart, product detail modals

### 2. Smart Packaging Selector (products.html)
- **Technology:** Custom JavaScript with price calculation
- **Features:** Visual packaging previews, bulk discount visualization
- **Options:** Retail boxes, wholesale crates, premium gift sets
- **Integration:** Real-time price updates, inventory checking

### 3. Seasonal Availability Calendar (products.html)
- **Technology:** ECharts.js interactive calendar
- **Features:** Month-based filtering, peak season indicators
- **Data:** Seasonal fruit availability with harvest periods
- **Functionality:** Click to filter products, educational tooltips

### 4. Shopping Cart & Checkout (all pages)
- **Technology:** Local storage with JavaScript management
- **Features:** Persistent cart, quantity adjustments, removal options
- **Checkout:** Streamlined form with multiple payment methods
- **Integration:** WhatsApp Business for order confirmation

## Visual Effects & Animations

### Background Effects
- **Aurora Gradient Flow:** Subtle animated background using CSS and JavaScript
- **Organic Patterns:** p5.js generated patterns inspired by fruit cross-sections
- **Parallax Elements:** Gentle background movement on scroll

### Text Effects
- **Typewriter Animation:** Typed.js for hero headlines
- **Color Cycling:** Anime.js for dynamic text color changes
- **Stagger Animations:** Sequential reveal of content sections

### Image Effects
- **Ken Burns Effect:** Subtle pan and zoom on hero images
- **Hover Transformations:** 3D tilt effects on product cards
- **Lazy Loading:** Progressive image loading for performance

### Scroll Animations
- **Reveal Effects:** Elements fade in when entering viewport
- **Stagger Timing:** Delayed animations for visual hierarchy
- **Parallax Limitations:** Maximum 8% translation for accessibility
